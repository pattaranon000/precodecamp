ข้อ 1.

  1. tag hr
  2. tag br
  3. tag hr ใช้ในการขีดเส้น tag br ใช้ในการขึ้นบรรทัดใหม่

ข้อ 2.

  1. tag ul
  2. tag ol
  3. tag ul(unordered list)คือ list แบบไม่กำหนดตัวเลข tag ol(ordered list)คือ list แบบกำหนดตัวเลข

ข้อ 3.

  1. tag select
  2. tag checkbox
  3. tag radio button
  4. tag select เป็น tag ตัวเลือก checkbox คือ tag เลือกได้หลายข้อ tag radio คือ tag ที่เลือกได้ตัวเลือกเดียว

ข้อ 4.

  1. tag div
  2. tag span
  3. tag div คือ ใช้ในการแสดงผลในแบบตาราง tag span คล้ายๆกับ tag div แต่ใช้ในการจัดข้อความสั้นๆ

ข้อ 5.

  1. tag script
  2. tag style
  3. script ใช้ในการแทรก javascript ลงใน html เพื่อเพิ่มลุกเล่น style คือการแต่งข้อความเนื้อหาต่างๆของเรา
